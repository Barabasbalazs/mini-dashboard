# Mini-Dashboard

MeiliSearch's next mini-dashboard. It will soon replace the current mini-dashboard available while lauching your MeiliSearch.

## Setup

```bash
yarn
```

or

```bash
npm install
```

## Run

```bash
yarn start
```

or

```bash
npm run start
```
